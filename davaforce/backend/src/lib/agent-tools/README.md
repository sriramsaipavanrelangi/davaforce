# Agent Tools Layer

This folder contains the agent tools that workforce agents should prefer over freeform database reasoning.

## Entry points

- Repo-level agent guide: `AGENTS.md`
- Local folder guide: `backend/src/lib/agent-tools/AGENTS.md`
- Import path: `backend/src/lib/agent-tools/index.ts`

## Modules

- `availabilitySearch`
- `capacityCalculator30_60_90`
- `skillsMatcher`
- `candidateScorer`
- `teamOptionBuilder`
- `riskAnalyzer`
- `ewaRecommendationBuilder`
- `explanationGenerator`
